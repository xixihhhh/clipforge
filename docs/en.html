05 / FAQ

FAQ

Is ClipForge free?

Yes — open source (AGPL-3.0), no watermark, unlimited videos. Free assets + free Edge TTS voices + free BGM + local FFmpeg: a full video with zero AI keys. Bring your own key for premium visuals (30+ models: GPT Image 2, Seedance 2.0, Kling 3.0…).

Will it distort my product photo?

No — image-to-image locks the original product. Backgrounds and lighting change; the product itself never does. What buyers see is what they get.

Do I need to appear on camera?

No. Fully faceless: product photo + AI visuals + voiceover + captions. One person can ship dozens of videos a day.

Which platforms does it export for?

TikTok Shop / Reels / Shorts 9:16, Xiaohongshu 3:4, Douyin, Kuaishou, WeChat Channels — one-click switching, plus hashtags, cover copy and engagement prompts.

How do I run it?

Download the Windows desktop app, or self-host the open-source Next.js web app (Docker supported). MCP server + CLI + agent skill included — Claude / Cursor can go from product link to video in one sentence.
