02 / WHY

不是又一个 AI 视频工具,专为带货而生

FIDELITY

商品保真,带货命门

image-to-image 锁定商品原图,换背景、打光也不改商品本体——绝不把你的产品 P 坏,避免「货不对板」的转化损失与售后风险。

SCRIPTS

会卖货的脚本

5 大品类深度模板 × 4 种带货风格(痛点种草 / 场景安利 / 对比测评 / 剧情)+ 黄金 3 秒钩子库,并自动产出话题标签、封面文案、互动引导。

ZERO-KEY

0 成本出整片

免费素材(Openverse 图 + Wikimedia 实拍视频)+ 免费 Edge TTS 配音(中英日韩西)+ 免费 BGM + 本地 FFmpeg 合成——没有任何 AI Key 也能出整片;要更高画质再自带 Key 接 30+ 模型。

COMPLIANT

合规默认开,发布不踩坑

AIGC 双标识对齐国标 GB 45438-2025、发布前限流自检逐项 ✓⚠✗、广告法违禁词即时高亮附改写建议——出片即合规,不用事后补。

BATCH

批量 + 爆款复刻

大促前 10 个商品一键批量出片;爆款脚本存模板一键套用,输入竞品链接换品重拍,A/B 多版本测转化。

GLOBAL

一条片,出海多语种

脚本一键翻译 + 对应语种免费音色重新配音:中文版发抖音,英文版发 TikTok,同源同画面。

FLYWHEEL

数据飞轮,学出爆款

回填播放 / 点赞 / 成交,按品类聚合出哪种脚本风格更能卖,直接回灌下一次生成——把「凭感觉」变成「用数据」。

AGENT-NATIVE

05 / FAQ

常见问题

ClipForge 是免费的吗?

是。开源(AGPL-3.0)、无水印、不限条数。免费素材 + 免费微软 Edge TTS 配音 + 免费 BGM + 本地 FFmpeg 合成,没有任何 AI Key 也能出整片;想要更高画质再自带 Key,一个接口聚合 7 大平台 30+ 模型(GPT Image 2 / Seedance 2.0 / Kling 3.0…)。

会把我的商品图改变形吗?

不会,这是命门功能:image-to-image 锁定商品原图,换背景、打光也不改商品本体——既保转化,也避免「货不对板」的合规与售后风险。

需要真人出镜吗?

不需要。商品图 + AI 配画面 + 免费配音 + 字幕自动合成,全程不出真人脸(faceless),一个人一天能出几十条。

支持发哪些平台?

抖音小店 9:16、快手、小红书 3:4、视频号、TikTok Shop / Reels / Shorts 规格一键切换导出,自动产出话题标签、封面文案、互动引导语,复制就能发。

国内发布会因为 AI 内容被限流吗?

合规默认开:成片自动写入 AIGC 隐式标识(对齐国标 GB 45438-2025),可一键烧显式「AI 生成」标识;发布前限流自检(风险词 / 钩子 / 时长 / 字幕 / CTA 逐项 ✓⚠✗ 附具体改法)+ 广告法违禁词扫描,出片即合规。

怎么部署使用?

两种方式:下载 Windows / macOS 桌面版双击即用;或自部署开源 Web 版(Next.js,可 Docker)。还内置 MCP Server + CLI + agent Skill,在 Claude / Cursor 里贴商品链接一句话出片。
